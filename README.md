# siteseed

[![CI](https://github.com/garricn/siteseed/actions/workflows/ci.yml/badge.svg)](https://github.com/garricn/siteseed/actions/workflows/ci.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Automated data seeding for web apps. Discovers an app's data model, generates realistic data, and seeds it via API calls or form filling.

Part of the site\* toolchain: [sitecap](https://github.com/garricn/sitecap) (capture) → [sitetest](https://github.com/garricn/sitetest) (test) → **siteseed** (seed) → sitefix (orchestrate).

## Install

```bash
npm install garricn/siteseed
```

## Quick Start

### Discover + seed in one step

```bash
siteseed run --openapi https://api.example.com/openapi.json --auto
```

### Step by step

```bash
# 1. Discover entities from an OpenAPI spec
siteseed discover --openapi https://api.example.com/openapi.json

# 2. Review / edit the generated seed plan
siteseed plan seed-plan.yaml

# 3. Execute the seed plan
siteseed run seed-plan.yaml
```

## Discovery Modes

**OpenAPI (preferred)** — reads an OpenAPI 3.x spec, extracts schemas, infers relationships from field names (`customerId` → `customers`), and builds a dependency graph.

**tRPC** — fetches router metadata from a tRPC endpoint to discover procedures and input schemas.

**GraphQL** — runs an introspection query to discover types, fields, and relationships.

**UI (fallback)** — uses [sitecap](https://github.com/garricn/sitecap) to discover forms from the accessibility tree, infers field types from input labels, and seeds by filling forms via [sitetest](https://github.com/garricn/sitetest).

## Seed Plan Format

Seed plans are YAML files describing what to seed:

```yaml
entities:
  - name: customers
    count: 5
  - name: orders
    count: 10
    overrides:
      status: active
```

Plans are generated by `siteseed discover` and can be edited by hand.

## Library API

```js
import { discover, seed } from "siteseed";

const graph = await discover({ openapi: "https://api.example.com/openapi.json" });
await seed({ graph, baseUrl: "https://api.example.com", counts: { customers: 5 } });
```
